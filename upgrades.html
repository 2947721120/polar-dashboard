Polar Update Tool (PUT)

You can download the latest version of PUT from the GitHub Releases page. The project is also open source (feel free to contribute).

Usage

PUT is a Java program, so it uses an executable JAR file. Double click it to open it (on Windows, Mac, or Linux). This is what you'll see when you open it (well, after you open it and choose a project folder, that is).

It will download the latest version of Polar directly from GitHub, and it will migrate newer changes into your project automatically.

Undoing Changes

It's possible something could go wrong during the upgrade process. If that's the case for you, there are two ways you can restore your project to how it was before...

The first way is a bit technical, but the easiest, for those who know how to use Git. Your project already has to be using Git, too.

The second way involves using the backup ZIP the upgrade tool created for you. The ZIP contains all of your original code and resource files.
